# 微软拼音自定义短语管理工具

## 简介

该工具用于管理微软拼音输入法的自定义短语（用户自定义词典），支持命令行和图形界面两种操作方式。  
提供导入/导出短语、列出短语、备份词库等功能。

---

## 命令行使用

### 导入短语

```bash
python pinyin_phrase_tool.py import <文件路径> [--force] [--dry-run]
```

- `--force`：覆盖已有短语时跳过确认提示。
- `--dry-run`：仅预览导入结果，不实际修改词库。

### 导出短语

```bash
python pinyin_phrase_tool.py export <文件路径>
```
导出当前所有自定义短语到指定的文本文件。

### 列出短语

```bash
python pinyin_phrase_tool.py list
```
在终端中显示所有已存在的自定义短语。

### 备份词库

未提供单独命令，可手动通过系统命令备份：

```bash
copy "%APPDATA%\Microsoft\InputMethod\Chs\ChsPinyinEUDPv1.lex" 备份目录路径
```

## GUI 使用

运行以下命令启动图形界面：

```bash
python pinyin_phrase_gui.py
```

### 功能按钮

| 按钮 | 功能 |
|------|------|
| 导入短语 | 选择 `.txt` 文件，导入自定义短语（过程中会询问是否覆盖已有项） |
| 导出短语 | 将当前所有短语保存到指定文本文件 |
| 列出短语 | 刷新中央文本框，显示当前所有短语（格式：拼音、序号、短语） |
| 备份词库 | 自动备份 `ChsPinyinEUDPv1.lex`，备份文件以时间戳命名 |
| 编辑 phrases.txt | 使用系统默认文本编辑器打开（或创建）`phrases.txt`，可手动增删改 |
| 关于 | 显示工具版本 |

### 其他界面元素

- **中央文本框**：实时显示短语列表，导入/刷新后自动更新。
- **底部标签**：左侧显示当前操作状态。

### 功能说明

- **导入**：支持两种格式行：`拼音 序号 短语` 或 `拼音 短语`（序号自动分配）。自动跳过注释行（`#` 开头）和格式错误的行。
- **导出**：将词库中所有短语按标准格式（`拼音 序号 短语`）导出为文本文件。
- **列出**：显示已存在的所有自定义短语。
- **备份**：修改词库前自动备份（导入时自动备份；GUI 中独立的“备份词库”按钮提供手动备份）。
- **编辑 phrases.txt**：提供一个可直接编辑的明文文件，方便用户自行添加或查看短语。注意：直接编辑该文件不会自动同步到词库，需通过“导入”功能导入修改后的文件才能生效。
- **修改生效**：修改词库后，需重新切换输入法（如按一次“中/英”切换键）或重新登录 Windows 才能使更改生效。
